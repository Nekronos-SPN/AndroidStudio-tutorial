## Welcome!

In this comprehensive codelab, you'll create your first Android app using the Java programming language. Throughout the codelab, we will use Android Studio, an Integrated Development Environment (IDE) based on JetBrains' IntelliJ IDEA, tailored specifically for Android app development.

:::caution
This codelab assumes you have experience with the [Java](https://www.java.com/en/) programming language. Basic knowledge of [Object Oriented Programming](https://en.wikipedia.org/wiki/Object-oriented_programming) will be necesary for the completion of this lab.
:::
